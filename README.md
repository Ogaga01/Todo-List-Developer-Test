# 📖 Todo List <a name="about-project"></a>

\*\*Todo List\*\* is a web application that helps you arrange your lists of tasks. Mark a task as complete and search for a particular task.

## 🛠 Built With <a name="built-with"></a>

- React
- TypeScript
- SASS

### Tech Stack <a name="tech-stack"></a>

- TypeScript
- SASS
